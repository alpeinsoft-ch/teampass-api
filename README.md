API for TeamPass
================

API for [TeamPass] passwords manager by [ALPEIN Software SWISS Team].

Documentation
-------------

Documentation is available at [teampass-api.readthedocs.org].


Installation
------------

Use [Composer] to install:

```
$ composer create-project alpeinsoft-ch/teampass-api
```
